# uix-charts 图表库

`uix-charts` 是基于 `Vue2.x` 和 `echarts4.x` 构建封装的图表组件库，用以解决繁杂的 `ECharts` 配置项以及数据转化带来的烦恼。`uix-charts` 生成一个 `ECharts` 图表时，用户只需关心**数据**与**配置项**，甚至无需配置项，即可生成一个默认的图表，使用 `uix-charts` 助你快捷、高效地构建图表。

# 安装

```JavaScript
npm i @xgfe/uix-charts echarts -S
```
